# Intelligent-Marine-Surveillance-System-for-Climate-Prediction-and-Ecosystem-Protection
OCEANGUARDIAN

**PROBLEM STATEMENT**

Rapid growth of marine renewable energy infrastructures such as tidal and 
offshore power plants

• These installations can cause environmental disturbances in ocean ecosystems

• Possible impacts include:

1.Heat entrapment in surrounding water

2.Artificial tidal disturbances

3.Changes in water chemistry and movement

• These disturbances can affect marine animal habitats, migration patterns, and 
biodiversity

• Current ocean monitoring methods are

1.Limited

2.Expensive

3.Mostly manual

4.Not real-time

• Lack of scalable and intelligent systems to monitor both

1.environmental changes

2.marine animal activity

**OBJECTIVE**

 Develop an autonomous marine monitoring system using sensor-based 
drones

• Monitor important ocean parameters such as

1.Temperature

2.pH

3.Water quality

4.Tidal movement

5.Electromagnetic disturbances

• ML Model 1: Identify marine animals near the drone (whale, dolphin, 
turtle)

• ML Model 2: Detect environmental disturbances (heat entrapment and 
tidal shifts)

• Enable real-time environmental monitoring using IoT communication

• Analyze collected data to predict climate-related ocean changes

• Provide a scalable and cost-effective ocean observation system

• Support marine ecosystem protection and sustainable ocean energy 
development
